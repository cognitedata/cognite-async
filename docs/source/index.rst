cognite-ds Documentation
========================

This is the documentation for cognite-ds.
This package is a work in progress, and includes more powerful functions for parallelization, data retrieval and machine learning.
Currently not intended for production code, interfaces may change at any time without warning.


.. contents::
   :local:

Installation
^^^^^^^^^^^^

To install this package, clone the repository and run:

.. code-block:: bash

   pip install .
